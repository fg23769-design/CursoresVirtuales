Gestión de Cursores Virtuales

Código Fuente


  

  

    
API Universal

    

      Endpoint: POST /cursor

      Parámetros:
      
    
        
  • pid: Identificador de proceso
    • 
        
  • accion: 'mover' o 'click'
    • 
        
  • x, y: Coordenadas (solo para mover)
    • 
      

      Ejemplo en Python:
      
import requests
requests.post("http://localhost:3001/cursor", json={
  "pid": 1234,
  "accion": "mover",
  "x": 100, "y": 200
})
      

      Ejemplo en JavaScript:
      
fetch("http://localhost:3001/cursor", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ pid: 1234, accion: "click" })
});
      

    

  

  

    
Ayuda y Configuración

    

      ¿Qué es CursoresVirtuales?

      Es una herramienta para crear y controlar cursores virtuales invisibles tipo hardware, útil para automatizaciones y pruebas.
    

    

      ¿Cómo usar?

      Desde la pestaña principal puedes elegir cuántos cursores crear y ver su estado.

      Controla los cursores desde cualquier lenguaje usando la API documentada en la pestaña "API".
